CARD GALLERY SETUP – Step-by-step UI and wiring
===============================================

1. CARDGALLERY CANVAS (if not present)
   - Open the CardGallery scene.
   - Right-click in Hierarchy → UI → Canvas. Rename it to "CardGallery_Canvas".
   - Ensure it has RectTransform, Canvas, Canvas Scaler, Graphic Raycaster (default for UI Canvas).

2. SCROLL VIEW
   - Right-click CardGallery_Canvas → UI → Scroll View.
   - Unity creates: Scroll View (with Scroll Rect) → Viewport → Content.
   - Rename "Content" to "Content" (keep name so script can find it if needed).

3. GRID LAYOUT ON CONTENT
   - Select the "Content" object (child of Viewport, inside Scroll View).
   - Add Component → Layout → Grid Layout Group.
   - Set:
     - Cell Size: e.g. 200 x 280 (match your card prefab size).
     - Spacing: e.g. 10 x 10.
     - Start Corner: Upper Left.
     - Start Axis: Horizontal.
     - Child Alignment: Upper Left.
     - Constraint: Fixed Column Count (e.g. 4) or Flexible (leave 0 for flexible).
   - On the Content RectTransform, add Content Size Fitter (optional):
     - Horizontal Fit: Unconstrained (or Preferred Size if you want width to drive).
     - Vertical Fit: Preferred Size (so scroll height grows with cards).

4. BACK BUTTON
   - Right-click CardGallery_Canvas → UI → Button - TextMeshPro (or UI → Button).
   - Rename to "BackButton". Set label text to "Back to Menu".
   - Position (e.g. top-left or bottom): use Anchor/Position so it stays visible when scrolling.

5. GAME OBJECT FOR CARD GALLERY MANAGER
   - Create Empty under CardGallery_Canvas (or at scene root). Name it "CardGalleryManager".
   - Add Component → CardGalleryManager (script).

6. INSPECTOR WIRING (CardGalleryManager)
   - Card Collection: Add elements and assign your CardData ScriptableObject assets (Assets → Create → Psyche → Card Data, or existing assets).
   - Card Prefab: Drag the same card prefab you use in SampleScene (the one with CardView).
   - Grid Content: Drag the "Content" object (the one with Grid Layout Group) here.
   - Auto Find Content: Leave checked if you didn’t assign Grid Content; script will find Content under the Scroll View.

7. BACK BUTTON → BACK TO MENU
   - Select BackButton. In Inspector, Button (Script) → On Click () → +.
   - Drag the "CardGalleryManager" GameObject into the object field.
   - Function: CardGalleryManager → BackToMenu().

8. MAIN MENU SCENE
   - BackToMenu() loads scene name "MainMenu". Either:
     - Name your main menu scene "MainMenu" and add it to Build Settings (File → Build Settings → Add Open Scenes), or
     - Change the script to load your actual menu scene name (e.g. "SampleScene" if menu is there).

9. BUILD SETTINGS
   - File → Build Settings. Ensure CardGallery and MainMenu (or your menu scene) are in "Scenes in Build".

Done. Enter Play and open CardGallery (e.g. from main menu View Cards); the grid should show all assigned cards and Back should load MainMenu.
